# 🛰️ CommBadge

Connecting Twitch communities with Discord, one broadcast at a time.

CommBadge is a smart Discord notification service designed to keep your community synchronized with Twitch. Whether it's the server owner going live, community members starting their own streams, or active chatters getting the recognition they deserve, CommBadge handles it automatically.

# 🚧 UNDER CONSTRUCTION
This project is currently under active construction; Once the service is live, more details about how to subscribe or set your own version up will be provided here.

# 🚀 Features

Streamer Alerts: Instantly notifies your Discord server the moment you go live on Twitch.

Community Support: Automatically posts a notification when a subscribed community member starts their stream, boosting peer-to-peer support.

Automatic Chat Shoutouts: If a community member is sends a message in your Twitch chat, CommBadge triggers an automatic shoutout to drive traffic to their channel.
